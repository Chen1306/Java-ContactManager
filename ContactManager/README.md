# 通讯录管理系统
## 面向对象程序设计课设

---

## 一、项目概述

本项目是一个基于 **Java + JavaFX** 的桌面通讯录管理应用程序，实现了课设要求的全部功能，包括联系人管理、分组管理、搜索（支持拼音声母模糊查询）、CSV/vCard 导入导出等。

---

## 二、技术架构

```
ContactManager/
├── pom.xml                          # Maven 构建配置
└── src/main/java/com/contactmanager/
    ├── App.java                     # 程序入口（JavaFX Application）
    ├── model/
    │   ├── Contact.java             # 联系人实体类
    │   └── ContactBook.java         # 通讯录数据模型（含搜索逻辑）
    ├── service/
    │   └── StorageService.java      # 数据存储服务（XML/CSV/vCard）
    ├── ui/
    │   ├── MainWindow.java          # 主界面（左侧分组 + 右侧列表）
    │   └── ContactEditDialog.java   # 联系人编辑对话框
    └── util/
        └── PinyinUtil.java          # 汉语拼音工具（声母/全拼转换）
```

**设计模式：**
- **MVC 模式**：Model（Contact/ContactBook）、View（MainWindow/Dialog）、Service（StorageService）分离
- **单例数据源**：ContactBook 作为唯一数据源，所有界面共享
- **观察者模式**：JavaFX Property 绑定，UI 自动响应数据变化

## 三、功能说明

### 3.1 联系人字段
| 字段 | 说明 |
|------|------|
| 姓名 | 必填，支持中英文 |
| 电话 | 固定电话 |
| 手机 | 手机号码 |
| 即时通信 | QQ/微信/Skype 等 + 账号 |
| 电子邮箱 | 邮箱地址 |
| 个人主页 | URL |
| 生日 | yyyy-MM-dd 格式 |
| 像片 | 选择本地图片文件路径 |
| 工作单位 | 公司/单位名称 |
| 家庭地址 | 详细地址 |
| 邮编 | 6 位邮政编码 |
| 所属组 | 可多选，支持跨组 |
| 备注 | 多行文本 |

### 3.2 搜索功能
支持以下模糊搜索方式：
- **姓名**：如输入「张」，匹配所有含「张」的姓名
- **手机/电话**：如输入「138」，匹配含该数字串的号码
- **拼音全拼**：如输入「zhang」，匹配张姓联系人
- **拼音声母**：如输入「zw」，匹配「张伟」（声母 z+w）
- **邮箱/单位**：同样支持模糊匹配

### 3.3 数据存储
- **主格式**：XML 文件（`~/ContactManager/contacts.xml`）
- **导出格式**：CSV（带 UTF-8 BOM，Excel 可直接打开）、vCard 3.0
- **导入格式**：CSV、vCard（兼容手机/QQ/163邮箱导出的 vcf 文件）

## 六、核心类说明

### Contact.java
联系人实体，实现 `Serializable` 和 `Comparable<Contact>`（按拼音排序）。
- `getFirstLetter()`：获取姓名首字母，用于分类显示
- `addGroup()` / `removeGroup()`：分组管理

### ContactBook.java
通讯录核心数据模型：
- `getContactsByGroup(String)`：按分组筛选，「全部」返回所有，「未分组」返回无组联系人
- `search(String)`：模糊搜索，调用 `matches()` 匹配各字段
- `removeGroup(String)`：删除分组时自动清除联系人的该分组标记

### StorageService.java
数据持久化服务：
- `save()` / `load()`：XML 格式读写
- `exportCSV()` / `importCSV()`：CSV 格式导入导出（UTF-8 with BOM）
- `exportVCard()` / `importVCard()`：vCard 3.0 格式，兼容主流手机

### PinyinUtil.java
内置常用汉字拼音表（覆盖百家姓及常用名字用字）：
- `toPinyin(String)`：转全拼（用于搜索匹配）
- `toPinyinInitials(String)`：转声母串（如「张伟」→「zw」）
- `updatePinyin(Contact)`：更新联系人拼音字段
